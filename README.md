# Bloom Space Web — сайт

Готовый статический сайт: `index.html`, `uslugi.html`, `ceny.html`, `privacy.html` + `assets/`.

## Как выложить на GitHub Pages (бесплатно)

1. Создайте репозиторий на github.com (например `bloomspaceweb`).
2. Закиньте туда СОДЕРЖИМОЕ этой папки (файлы должны лежать в корне репозитория, не во вложенной папке).
   Через VS Code: откройте эту папку → вкладка Source Control → Publish to GitHub.
3. В репозитории: Settings → Pages → Source: «Deploy from a branch» → Branch: `main`, папка `/ (root)` → Save.
4. Через 1–2 минуты сайт будет доступен по адресу `https://<ваш-ник>.github.io/<имя-репозитория>/`.

## Что уже сделано
- Форма заявки шлёт данные в вашу Google Таблицу (Apps Script URL зашит в index.html).
- Валидация формы (пустые поля не отправляются) + honeypot против спам-ботов.
- Instagram / почта — в футере.
- Видео в шапке: `assets/hero-full.webm` (~17 МБ). Если захотите ускорить загрузку — пережмите в mp4 H.264 ~8 Мбит/с и замените src.

## Безопасность
Сайт статический: нет сервера, базы и логинов — взламывать нечего. Единственный публичный элемент — URL Google-скрипта (максимальный риск: спам-строки в таблице; лечится перевыпуском деплоя скрипта).
